# Installation

## Requirements

Os-level requirements:
- PostgreSQL

Install Python requirements with pipenv:

```python
pipenv install
```

Or the old fashioned way, with pip:

```python
pip install -U -r requirements.txt
```

## Set up

Make a Postgres database:

```
createdb karaoke
```
